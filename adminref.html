General notes

  1. Admins bypass the information command rate limit.
  2. Admins can use all bet commands a streamer can use, refer to the regular command reference for them.
  3. The commands in the “Debugging” section can only be used while debug mode is enabled.

Bets

!bet results – show bet results

!bet payout – pay out bet

!bet changetime time – change the bet time

!bet forcereset – reset the bet to “open” status (usage unlimited in others’ channels)

!bet forceenter user time – enter user into a running bet

Godimages

!godimage queue – show pending godimage requests

!godimage check ID – check request for waifu

!godimage acceptsingle ID – accept request for waifu (only affects their card)

!godimage reject ID reason – reject request; reason is required

Set badge images

!setbadge queue – show pending setimage requests

!setbadge check ID – check request for set

!setbadge accept ID – accept request for set

!setbadge reject ID reason – reject request; reason is required

Miscellaneous

!announce message – send message to all channels the bot has joined

!givefreepack user booster [amount] – give one or more free packs

!raritychange ID rarity – change rarity of a card

Bot Management

!nepjoin channel – make the bot join the given channel (do not specify #)

!nepleave channel – make bot leave the given channel (do not specify #)

!reload – reload bot configuration

!quit – shut down bot

!import URL – import cards from given URL

Giveaways

!giveaway open – open a new giveaway

!giveaway pick winners – pick winners, at least 1

!giveaway close – finish the giveaway

Raffles

!raffle open – open a new raffle

!raffle close – close raffle entries to start drawing

!raffle drawgrand – draw grand prize winner

!raffle draw winners – draw winners

!raffle complete – finish the raffle

Debugging

!as user command – execute command as another user

!debug – changes title and game on marathon channel to testing values
